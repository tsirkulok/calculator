# 🧮 Console Calculator (Node.js Pet-Project)

Це інтерактивний консольний калькулятор, побудований на базі **Node.js**. Проєкт створений для відпрацювання навичок роботи з асинхронним програмуванням, обробкою вводу користувача та валідацією даних.

## ✨ Особливості (Features)
* **Асинхронний інтерфейс:** Використання `readline/promises` для плавного введення даних без "пекла колбеків".
* **Розумна валідація:** Перевірка надійності введених чисел за допомогою `isNaN()`.
* **Безпека обчислень:** Вбудована перевірка на ділення на нуль.
* **Циклічність:** Програма дозволяє виконувати кілька розрахунків поспіль, не завершуючи роботу після першої помилки або операції.
* **Чиста архітектура:** Математичні операції винесені в окремі функції для зручності тестування.

## 🛠 Технологічний стек
* **Мова:** JavaScript (Runtime: [Node.js](https://nodejs.org/))
* **Модулі:** `node:readline/promises`
* **Логіка:** Async/Await, Switch/Case, Try/Catch logic.
